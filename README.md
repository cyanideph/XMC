# XMC Construction Landing Page

This is a modern parallax landing page for XMC Construction, a construction company based in Anahawan, Southern Leyte.

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

The easiest way to deploy your Next.js app is to use the [Vercel Platform](https://vercel.com/new?utm_medium=default-template&filter=next.js&utm_source=create-next-app&utm_campaign=create-next-app-readme) from the creators of Next.js.

To set up this project:

1. Create a new repository on GitHub.
2. Clone the repository to your local machine.
3. Create the folder structure and files as outlined above.
4. Copy the provided code into each respective file.
5. Run `npm install` to install the dependencies.
6. Replace the placeholder images in the `public/images/` folder with your actual project images.
7. Customize the content, colors, and other details in the `LandingPage.tsx` file to match XMC Construction's branding and services.
8. Run `npm run dev` to start the development server and view your landing page.

This structure provides a complete setup for a Next.js project with TypeScript, Tailwind CSS, Framer Motion, and other necessary dependencies for the modern parallax landing page. You can now push this to your GitHub repository and start customizing the content for XMC Construction.
